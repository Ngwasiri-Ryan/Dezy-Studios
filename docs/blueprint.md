# **App Name**: Dezy Arts Portfolio

## Core Features:

- Service Showcase: Display all provided services such as Photography, Graphic Design, and Product Design.
- Category-Based Portfolio: Enable filtering the portfolio by service type (Photography, Video, Products, etc.).
- Contact Form: Allow clients to send messages via a contact form.
- Product Display: Showcase products with images, descriptions, and pricing, with instructions to contact via WhatsApp or phone for purchases.
- Client Testimonials: Display client feedback through a mix of text and video/visuals.
- AI-Powered Tag Generator: Analyze uploaded images or videos using AI to automatically suggest relevant tags and descriptions for portfolio items. The AI tool uses object recognition to extract relevant features.

## Style Guidelines:

- Primary color: Deep Onyx (#1A1A1A) for typography, logo, and icons.
- Secondary color: Dusty Rose (#E5B8AD) for section backgrounds and subtle borders.
- Neutral color: Pearl White (#FDFBFB) for the main site background.
- Accent color: Sunset Pink (#D81B60) for buttons, links, and active states.
- Headings font: 'Playfair' (serif) for an elegant, high-contrast, boutique vibe.
- Body text font: 'Inter' (sans-serif) for clean, geometric, easy-to-read text. Note: currently only Google Fonts are supported.
- Navigation bar: Top horizontal menu with logo on the left and links on the right, sticky with a slide-down effect and glassmorphism (semi-transparent Pearl White background with subtle blur).
- Project cards: Video playback on hover to showcase skills.
